🟦 ONE-AIR スマホ側運用マニュアル（正式版）

― 審査官 / 現場スタッフ向け ―
ver.1.0（2025-11-14）

1. ONE-AIR とは

ONE-AIR は、入国者の 一次審査をスマホで行うための補助システム です。
スマホに何もインストールする必要はありません。

ブラウザだけで動作

完全ローカル（オフライン）

個人情報はスマホに残らない

安全性と簡便性を両立した、臨時運用向けシステムです。

2. スマホ側で必要なもの

ブラウザ（Chrome / Safari / Edge など）

Wi-Fi（PCのホットスポットに接続）

※ カメラ不要
※ アプリ不要
※ 専用端末でなくても利用可能

3. PC側が準備するもの

審査官（または補助係）が PC で：

ONE-AIR USB 挿入

oneair_launcher.bat を実行

ランチャー画面を表示

PCホットスポット（ONE-AIR-AP）を ON

ここまでがスマホ接続前提となる準備です。

4. スマホの接続手順（最重要）
STEP 1：Wi-Fi を ON

スマホの Wi-Fi 設定から ONE-AIR-AP を選ぶ。

SSID: ONE-AIR-AP
パスワード: PC画面に表示されたもの

STEP 2：ブラウザを開く

Chrome / Safari / Edge など、任意のブラウザ。

STEP 3：PCが表示した URL をそのまま入力

PCのランチャー画面に表示される：

http://oneair.local:8800/<短縮コード>


例：

http://oneair.local:8800/JG5Q2D


これをブラウザのURL欄に入力。

STEP 4：自動で ONE-AIR UI が表示される

画面が自動で転送され、一次審査画面が開きます。

（1台ずつ自動割当。PC側では何も操作不要）

5. mDNS が使えない場合（対処）

環境によっては oneair.local が解決できない ことがあります。（機種差/設定差）

その場合は、PC画面に表示される PC-IP を使って接続してください。

例：

http://192.168.137.1:8800/JG5Q2D


※ この代替URLは PCランチャー画面で必ず確認できます。

6. スマホでできること
6.1 入力項目一覧

パスポート MRZ 入力

渡航目的の選択

危険度（リスク）判定の確認

二次審査（ボタン1つ）

メモ（審査官コメント）

入力した情報は リアルタイムで USB に保存 されます。

6.2 操作の流れ

MRZ入力

内容確認

二次審査の判断

提出 → 終了

次の入国者へ（URL再入力不要）

スマホ側の操作はすべて 直感的 UI にしています。

7. トラブル時の対処
7.1 「サイトが見つかりません」

Wi-Fiが ONE-AIR-AP か確認

oneair.local ではなく oneair.local:8800 を入力しているか確認

代替URL（PC-IP版）を使用：

http://<PC-IP>:8800/<コード>

7.2 ブラウザが固まった

一度閉じて、URLだけ再入力

コードの有効期限が切れた場合は、PC側が新しいコードを表示します

7.3 画面が真っ白

Wi-Fi接続を切 → 再接続

それでもダメなら PC側（LANCHAER）が新しいコードを出すので、再入力

8. 情報保護について

スマホには何も保存されません

PCにも情報は保存されません（USBのみ）

オフライン運用のため外部送信は不可能

ISMS・情報保護の要件に完全準拠しています。

9. 終了手順（スマホ側）

Wi-FiをOFF

ブラウザを閉じる

端末返却（必要に応じて）

以上で完了です。

10. 一枚でわかるまとめ
① Wi-Fi → ONE-AIR-AP 接続
② ブラウザを開く
③ http://oneair.local:8800/<コード>
④ 審査開始


（代替）

http://<PC-IP>:8800/<コード>
