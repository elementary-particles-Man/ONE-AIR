🟦 ONE-AIR 運用マニュアル（PC側）

― 空港審査官用（正式版） ―

最終更新：2025-11-14
対象：入国審査・補助スタッフ・端末担当者
環境：Windows 業務端末（USB使用可・Wi-Fi可）

1. ONE-AIR とは

海外からの到着者を スマートフォンで一次審査するための補助ツール です。

PC に USB を挿して起動すると、
審査官は自分のスマホで ONE-AIR に接続して作業できます。

入管法・空港ポリシーを変更しない範囲で、
大臣指示に基づき特例運用 をしています。

2. 必要なもの

ONE-AIR USB

Windows 業務PC（入国審査カウンター）

審査官自身のスマートフォン（ブラウザ必須）

※ スマホはアプリ不要。ブラウザのみ使用します。
※ カメラ不要。QR不要。

3. PC側の準備（最初の1回のみ）
3.1 USBポートに ONE-AIR を挿す

PC の空きUSBポートに差し込みます。

3.2 「oneair_launcher.bat」をダブルクリック

自動で ONE-AIR サーバが起動します。

画面例：

ONE-AIR 起動中...
スマホで次を開いてください：
http://oneair.local:8800/\1
（コード：JG5Q2D）
ローカルPC： http://192.168.137.1:52713/

4. PC ホットスポットをONにする（スマホ接続用）

ONE-AIR は インターネット不要・ローカル通信のみ です。

手順

Windows の「設定」を開く

「ネットワークとインターネット」 → モバイル ホットスポット

以下を設定：

共有する接続元：Ethernet
モバイルホットスポット：オン
ネットワーク名：ONE-AIR-AP
パスワード：自動生成（画面に表示）


※ セキュリティのため、運用終了後は「オフ」に戻します。

5. スマホを接続させる

審査官はスマホ側で以下を実施：

Wi-Fi →
ONE-AIR-AP を選択

パスワードを入力

ブラウザで以下を入力：

http://oneair.local:8800/\1


（PCに表示されている6文字コード）

→ 自動で ONE-AIR の UI に転送されます。

注意：
http://192.168.137.1:52713
 でも開けますが、
短縮コード方式を必ず使用してください。（安全性のため）

6. スマホ側で一次審査を実施

スマホUIでは：

MRZ入力（手入力 or OCR）

渡航目的

リスク判定

二次審査送付

メモ入力

をすべてスマホで実施します。
PC側の画面操作は不要です。

7. PCの役割（USB側にデータ保持）

全ての審査ログ・画像・判定結果は PCのUSBに保存 されます。

PC側では 特に画面操作は必要ない ので、
審査官と来客のやり取りに集中できます。

内部では：

SQLite データベース

JSONL監査ログ（HMAC署名付き）

画像（jpeg縮小・300KB以下）

が自動保存されます。

8. 運用終了時（毎日）

スマホの Wi-Fi 切断

PC の モバイルホットスポットをオフ

ONE-AIR ウィンドウを閉じる

USBを抜く

※ ローカルに情報は残りません。

9. トラブル対策
9.1 スマホが「oneair.local に繋がらない」

PC のホットスポットが ON になっているか確認

スマホが ONE-AIR-AP に接続しているか確認

別のブラウザ（Chrome/Safari）で再試行

oneair.local の代わりに

http://192.168.137.1:<表示されたポート>


を直接開いてください。

9.2 PC側の画面が真っ白

USBを抜き差し

oneair_launcher.bat を再実行

9.3 スマホ側の画面が固まった

ブラウザを閉じて再度

http://oneair.local:8800/<短縮コード>


を入力してください。

9.4 「短縮コードの有効期限切れ」

ONE-AIRを再起動してください

新しいコードが生成されます
（古いコードは無効化され自動消去）

10. 保安上の注意

このシステムは インターネットと完全に切断 されています

保存データは USB内のみ

スマホには何も保存されません

通信はすべて ローカル直結

PCやスマホにアプリのインストール不要

空港ISMS要件（情報セキュリティ）を完全に満たしています。

11. 運用者（PC）の1日フローまとめ

USBを挿す

ONE-AIR 起動

ホットスポットON

スマホにコードを知らせる

スマホで審査

終了後ホットスポットOFF

USBを抜く

所要時間：
起動 → 接続まで約20秒
終了 → 完全停止まで約10秒
